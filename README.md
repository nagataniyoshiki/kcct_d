# 神戸高専 電子工学科 実験資料


## 概要 / Outline

神戸市立工業高等専門学校 電子工学科の学生実験で使用している資料やソースコードなどを掲載しています。

### openmp
5年生の実験「各種計算ハードウェアの活用（D5）～CPUの性能を引き出す手法～」の資料とソースコードです。

### cuda
5年生の実験「各種計算ハードウェアの活用（D5）～GPU（GPGPU）で並列演算をおこなう手法～」の資料とソースコードです。
